Como criar um repositório usando github

Passos para criar um repositório de projeto usando Git

  1. Crie o repositório no GitHub: clique em "New" e preencha nome, descrição e visibilidade.
  2. No seu projeto local, inicialize o repositório Git: 
    git init
  3. Adicione arquivos e crie o primeiro commit: 
    git add .
git commit -m "Initial commit"
  4. Defina o branch principal (opcional): 
    git branch -M main
  5. Adicione o remote origin (substitua a URL pelo endereço do repositório GitHub): 
    git remote add origin https://github.com/USERNAME/REPO.git
  6. Envie (push) o commit para o GitHub: 
    git push -u origin main
  7. No GitHub, verifique os arquivos e ajuste `README.md`, `.gitignore` e configurações do repositório.
